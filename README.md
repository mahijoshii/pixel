# Pixel

An STM32-based autonomous obstacle-avoidance robot built on the Nucleo-F401RE.

## Overview
This robot uses an ultrasonic sensor to detect obstacles and controls two DC motors through a motor driver to navigate autonomously.

## Hardware
- STM32 Nucleo-F401RE
- HC-SR04 ultrasonic sensor
- L298N motor driver
- 2 DC motors
- external battery pack
